# R and RStudio: Information, Tutorials, and Workshops

[R](https://www.r-project.org/) is a programming language and a free software environment to write and execute the R language. [RStudio](https://www.rstudio.com/) is an Integrated Development Environment that provides free and open\-source tools for R. While you can run your data analysis in the basic R environment, RStudio has a more intuitive interface and more tools to help you write your R code.

Getting started
---------------

* [Download R](http://cran.r-project.org/mirrors.html)
* Download [RStudio](https://www.rstudio.com/products/rstudio/download/#download)
* Follow this [guide](http://mdl.library.utoronto.ca/technology/tutorials/getting-started-r-and-r-studio) for additional instructions to get started with R and RStudio
* See below for course, workshop and tutorials to get an introduction to R
